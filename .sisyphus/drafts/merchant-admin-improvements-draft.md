## 现状诊断

### 商家端 (frontend-merchant)

**已有功能**：
- 商户工作台（店铺选择、数据统计）
- 门店创建（基本信息+图片上传）
- 优惠券上架（平价券+秒杀券+时间预设）
- 菜品管理（增删改查）
- 订单管理（筛选+核销）
- 我的门店列表

**关键缺失**（按严重性）：
| 问题 | 位置 | 影响 |
|------|------|------|
| 无登录/登出界面 | 缺少Login.vue | 无法独立使用 |
| 缺少统一错误处理 | api/client.js | 错误提示混乱 |
| 无加载状态 | 所有列表 | 体验差 |
| 表单无验证反馈 | 所有form | 用户不知道错在哪 |
| 图片上传无预览 | MerchantHome.vue | 操作不确定 |
| 订单状态显示数字 | MerchantHome.vue:134 | 用户看不懂 |
| 无空状态设计 | 列表区域 | 视觉不友好 |
| console.log未清理 | MerchantHome.vue:222 | 生产环境问题 |

### 管理端 (frontend-admin)

**已有功能**：
- 运营看板（今日订单、待审统计）
- 订单检索+导出CSV
- 用户管理（启用/禁用）
- 商户审核（通过/拒绝）
- 店铺审核（通过/拒绝）
- 内容审核（帖子审核）
- 系统配置（本地草稿）
- 审核记录页面（AuditRecords.vue）

**关键缺失**（按严重性）：
| 问题 | 位置 | 影响 |
|------|------|------|
| 无登录/登出界面 | 缺少Login.vue | 无法独立使用 |
| 审核操作无确认 | AdminHome.vue:99-116 | 误操作风险 |
| 拒绝原因硬编码 | AdminHome.vue:289 | 无法自定义 |
| 系统配置仅本地 | AdminHome.vue:326-329 | 不生效 |
| 统计看板数据不完整 | AdminHome.vue:316-324 | 缺少趋势 |
| 订单筛选页码丢失 | AdminHome.vue:185-191 | 用户困惑 |
| 加载状态缺失 | 所有异步操作 | 体验问题 |

## 技术债务

1. **路由不完整**：两个项目都只有首页路由
2. **缺少统一拦截器**：401/403未处理
3. **缺少工具函数**：时间格式化、状态映射重复
4. **样式重复**：两个项目各自维护相同样式
5. **代码审查**：存在console.log和硬编码文本

---

## 约束与假设

- **无外部UI库**：保持纯CSS变量方案
- **Vue3 Composition API**：继续使用`<script setup>`
- **API网关不变**：保持localhost:8081
- **存储方案不变**：继续使用localStorage
- **认证机制不变**：继续使用JWT + X-Refresh-Token
